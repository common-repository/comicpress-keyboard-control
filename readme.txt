== Description ==

This plugin adds simple javascript that switches between comic entries in Comicpress. It was designed for Comicpress 2.9.

== Installation ==

1. Upload `comicpress-keyboard-control.zip` in the 'Plugins' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy world domination
